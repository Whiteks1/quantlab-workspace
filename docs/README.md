# QuantLab Research Local Docs

This folder keeps the minimum local documentation needed to understand the public QuantLab Research landing without depending entirely on the upstream QuantLab repository.

These docs are local mirrors or condensed adaptations of the upstream QuantLab material. They should stay aligned with the landing copy and the public surface presented in this repo.

## Docs index

- [brand-architecture.md](./brand-architecture.md): canonical scope rule for this repo as the public QuantLab Research surface
- [architecture.md](./architecture.md): product layers, authority model, and why `QuantLab Research` exists as a public surface
- [cli.md](./cli.md): compact guide to the command families the landing references
- [run-artifact-contract.md](./run-artifact-contract.md): canonical artifact roots and file contracts
- [output-structure.md](./output-structure.md): quick reference for the output paths exposed in the landing

## Scope

These files are intentionally selective.

- They document the concepts the landing links to directly.
- They do not attempt to mirror the full upstream QuantLab documentation tree.
- They should remain readable to someone who only has access to this repo.
