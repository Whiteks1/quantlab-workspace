# Desktop evidence convention

Store real desktop captures in this folder using stable names so the public workspace can point at them without changing copy or layout.

Current status:

- filename contract defined
- no real captures committed yet

Recommended filenames:

- `workspace-overview.png`
- `runs.png`
- `run-detail.png`
- `artifacts.png`
- `validation.png`

Guidelines:

- Prefer PNG for interface captures.
- Keep the captures consistent in theme and zoom level.
- Replace provisional assets one by one instead of adding parallel variants.
- If a new surface becomes canonical, update the README and the landing component that references it.
